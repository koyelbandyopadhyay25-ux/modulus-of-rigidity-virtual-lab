# Interactive Virtual Physics Lab: Modulus of Rigidity by Dynamic Method

A comprehensive, web-based interactive physics laboratory designed to simulate the determination of the modulus of rigidity of a metallic wire using the dynamic method (torsional pendulum). 

This platform provides engineering students with an accessible, realistic digital environment to perform experiments, enhancing conceptual understanding and practical laboratory learning without the need for physical apparatus.

## ✨ Features

* **Interactive Apparatus Setup:** Manipulate wire and cylinder dimensions (length, radius, mass) using responsive sliders with realistic physical constraints.
* **Real-time Simulation:** Visual torsional pendulum animation that dynamically scales its oscillation frequency based on the physical parameters chosen.
* **Automated Data Collection:** Simulates experimental noise and automatically populates observation tables with 30-oscillation time trials.
* **Dynamic Graphing:** Auto-generates an Angle vs. Time damped oscillation graph using Chart.js.
* **Complex Mathematical Rendering:** Integrates MathJax to beautifully display complex physics formulas, calculations, and exact error analysis derivations.
* **Modern UI/UX:** Features a professional academic interface with responsive pill-shaped tabs, an expandable contributors accordion, and a seamless Dark/Light mode toggle.

## 🛠️ Technologies Used

* **Frontend:** HTML5, CSS3
* **Logic & Simulation:** Vanilla JavaScript (ES6)
* **Data Visualization:** Chart.js
* **Typography & Math:** MathJax, Google Fonts (Inter)
